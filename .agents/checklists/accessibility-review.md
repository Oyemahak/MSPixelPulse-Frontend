# Accessibility Review Checklist

- [ ] Semantic structure
- [ ] Keyboard access
- [ ] Focus states
- [ ] Labels and errors
- [ ] Contrast and reduced motion
- [ ] Every icon-only button or link has a contextual accessible name; `title` is not the accessible-name substitute
- [ ] Decorative icons are hidden from assistive technology
- [ ] Toggles expose `aria-pressed` and disclosure controls expose `aria-expanded`
- [ ] Non-submit buttons declare `type="button"`

## Evidence Required
List files inspected, commands run, manual checks, screenshots or URLs where relevant, unresolved risks, and reviewer.
