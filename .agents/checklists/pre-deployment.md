# Pre Deployment Checklist

- [ ] Build passes
- [ ] Tests pass or gaps listed
- [ ] Env vars verified by name only
- [ ] Rollback noted
- [ ] Approval received
- [ ] `/`, `/login`, `/register`, and an invalid route pass shared-shell smoke tests
- [ ] Both-theme screenshots or browser evidence are recorded for affected routes
- [ ] Responsive matrix, accessible icon names, reduced motion, blur fallback, and representative role checks pass

## Evidence Required
List files inspected, commands run, manual checks, screenshots or URLs where relevant, unresolved risks, and reviewer.
