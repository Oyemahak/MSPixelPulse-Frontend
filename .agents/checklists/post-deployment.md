# Post Deployment Checklist

- [ ] Production health checked
- [ ] Key flows smoke-tested
- [ ] Console/network errors checked
- [ ] Monitoring notes captured
- [ ] Handoff complete
- [ ] Production `/`, `/login`, invalid route, and representative admin/developer/client routes checked
- [ ] Theme persists after reload and navigation
- [ ] SPA deep links refresh successfully

## Evidence Required
List files inspected, commands run, manual checks, screenshots or URLs where relevant, unresolved risks, and reviewer.
